Setup
------

1.  Download xampp	https://www.apachefriends.org/download.html
2. Install to c:\ directory
3.  After Installation start the following through  XAMPP console
	a. mySQL
	b. Apache
	c. Tomcat *currently not being used, is needed for jsp intergration*
4. Navigate to localhost/phpmyadmin or 127.0.0.1
	a. Username is "root" password is blank
	b. This is where any database management will occur

Creating the Local Database
---------------------------

1. Start XAMPP control panel
     a. Turn on Apache/mySQL
2. In browser navigate to
     a. localhost/phpmyadmin
     b. 127.0.0.1/phpmyadmin
3. On left navigate pane select new
     a. Enter databse name "Career Services"
     b. Press create
     c. Select DB career services in left navigation pane
     d. Create new table named "db_login"
     e. Navigate to the SQL tab
     f. Paste following SQL into area and press go
     
     CREATE TABLE `db_login` (
     `username` varchar(255) NOT NULL,
     `password` varchar(32) NOT NULL
     ) ENGINE=InnoDB DEFAULT CHARSET=latin1;

     g. create new table "student_form"
